# 项目启动命令说明

## 1. 后端服务启动

后端基于 Django 框架构建，强烈建议在独立的虚拟环境中运行：

### 1.1 创建与激活虚拟环境（如未创建）

**使用 Python 内置的 venv（推荐）:**
1. 进入后端目录：
   ```bash
   cd backend
   ```
2. 创建名为 `venv` 的虚拟环境：
   ```bash
   python -m venv venv
   ```
3. 激活虚拟环境：
   - **Windows:**
     ```bash
     venv\Scripts\activate
     ```
   - **Mac/Linux:**
     ```bash
     source venv/bin/activate
     ```
4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

*(如果您使用 Miniconda，请使用 `conda create -n virtual_human python=3.10` 创建环境，并用 `conda activate virtual_human` 激活后再安装依赖。)*

### 1.2 启动 Django 服务

1. 确保已在 `backend` 目录下，并且**虚拟环境处于激活状态**（终端前会显示 `(venv)` 或您的 conda 环境名）。
2. 启动 Django 开发服务器：
   ```bash
   cd backend
   ```
3. 启动 Django 开发服务器：
   ```bash
   python manage.py runserver
   ```
   *默认情况下，后端服务将运行在 `http://127.0.0.1:8000/`*

---

## 2. 前端服务启动

前端基于 Vue 3 + Vite 构建，请按照以下步骤启动：

1. 打开一个新的终端窗口。
2. 进入前端目录：
   ```bash
   cd frontend
   ```
3. （如果是首次运行或刚刚拉取代码）安装 Node 依赖：
   ```bash
   npm install
   ```
4. 启动 Vite 开发服务器：
   ```bash
   npm run dev
   ```
   *启动成功后，终端会打印出本地访问地址，通常为 `http://localhost:5173/`*
